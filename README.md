DAO Framework
=======

An implementation of [DAO pattern](http://www.oracle.com/technetwork/java/dataaccessobject-138824.html) for SQLite and JSON.

Inspired by [RedMadRobot's DAO](https://github.com/RedMadRobot/DAO).

## Requirements

- XCode 9.3
- Swift 4.1
- iOS 10
